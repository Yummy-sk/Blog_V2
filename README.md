# Notion API 기반 염상권 기술 블로그

Notion API를 사용하여 파일을 업로드 하는 방식이 아닌 Notion으로 간편하게 글을 작성할 수 있는 블로그입니다.

## 사용한 기술

- TypeScript
- React.js
- Next.js
- Styled Components
- Notion API
